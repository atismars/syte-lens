Our privacy promise

A full analysis only runs when you click the icon (or if you enable auto-scan). To color the icon for sites already rated, we do a lightweight, anonymous domain-only lookup — no page content, no accounts, and we never build a profile of your browsing or sell your data. You can turn that lookup off below.

Exactly what we do (and don't)

  • We send: the site's root domain (e.g. example.com), and for the page you're actively checking, its title, meta description, and a short text sample.
  • We never send: the full URL, page paths, query-string values, form data, or anything you type. Only the names of ad-tracking parameters are noted.
  • No identity: no accounts, no cookies, no user IDs. The verdict cache is keyed by domain and shared by everyone — it can't be tied back to you.
  • Known-site coloring: to color the icon, we send only the root domain to check our ratings database (no page content, no analysis). Toggleable below.
  • No history: we don't build or store a per-user record of where you browse.
  • Open source: the client is fully auditable on GitHub — read every line that runs in your browser.

Settings

Show ratings for already-rated sites
On by default. Colors the toolbar icon (green / amber / red) for sites already in our shared ratings database — a lightweight, anonymous domain lookup. No page content is sent and no new analysis runs.
Auto-scan sites as I browse
Off by default. When on, Sytelens analyzes every site you visit so the toolbar icon pre-colors — which means each domain you visit is sent for analysis without clicking. Leave off to keep the click-only promise above.
